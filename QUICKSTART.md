# FILETREEFORGE v2.0 - Quick Start Guide

Get up and running with FILETREEFORGE in 5 minutes!

## Installation (1 minute)

```bash
cd filetreeforge
npm install
npm run compile
```

Press `F5` in VS Code to launch Extension Development Host.
